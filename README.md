# E-Commerce Shipping Analysis Dashboard

## Overview
This project is an interactive Excel dashboard created to analyze e-commerce shipping performance and delivery delays.

## Objectives
- Analyze on-time vs late deliveries
- Study the impact of shipment mode on delivery performance
- Compare warehouse-wise delivery performance
- Evaluate how product importance affects delivery outcomes

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Dashboard Features
- KPI cards for Total Orders, On-Time Delivery %, Late Delivery %, Average Customer Rating, and Average Customer Calls
- Interactive slicers for Product Importance, Warehouse, and Shipment Mode
- Charts showing delivery performance across different business factors

## Key Insights
- Around 60% of deliveries are late
- Shipment mode has limited impact on delivery performance
- High importance products show higher late delivery percentage
- Warehouse performance is relatively consistent across locations
